Basemaps

A basemap is a reference layer that displays underneath all other layers. More details are displayed as the map is zoomed in. The identify tool will not identify features on a basemap layer.

To change the basemap:

  • Click on a basemap icon and the map will update immediately to the new basemap
  • The basemap can be changed at any time.



Add Layers

Select layers of interest to add to the map. The map layers are organized thematically into groups.

  • Click a group name to expand or collapse its layer list
  • Check the box next to the layer’s name to add it to the map (uncheck the box to remove the layer)
  • Click a layer name to view a short description and access metadata and download links.

 




Active layers

Provides tools to manage the active layers in the map

  • Click the “X” to the left of the checkbox to remove the layer from the list. 
  • Check/uncheck the box for layer visibility.
  • Use the slide bar to adjust the layer opacity.
  • Use the up/down arrows to control the layer stacking order in the map.
  • Click a layer name to view a short description and access metadata and download links.



Reset Map

It resets the map back to the default settings for the user to create a new map. It clears all layers, zoom scale, and basemap.

  • Click the tab to reset the map



Identify

Identify features on the map. The identify tool will not identify features on a basemap layer.

  • Click a feature in the map to identify its attributes.



Draw Tool

Use the draw tool to add basic shapes, buffer circles, and add text to the map. Once added, graphics can be selected to move, edit, or delete.

To add a Shape:

  • Select the graphic color
  • Select one of the basic shapes from the dropdown menus
  • Hover the mouse over the map and follow the tool tip instructions to place the graphic. 

To add Text: 

  • Select the graphic color
  • Enter custom text in the dialog box
  • Click the “Add Text” button
  • Click on the map to place the text.

Add a Buffer:

  • Select the graphic color
  • Enter the buffer distance (radius) and select the units
  • Click the “Buffer” button
  • Click on the map to place the graphic.

Move a graphic object:

  • Click on the graphic object in the map
  • Click and Drag the object to a new location

Edit the shape/size of a graphic object (excluding text):

  • Click on the graphic object in the map
  • Click and Drag the objects handles
  • To deselect the graphic object click in the map away from a graphic object




Measurement Tool

Use to measure areas, distances, or point locations in a variety of units.

Measure Area:

  • Click the “Area” button
  • Click the map to begin
  • Continue clicking on the map to outline an area and double click to finish
  • Results are given in acres
  • Use the dropdown box to select other units

Measure Distance:

  • Click the “Distance” button
  • Click the map to begin
  • Continue clicking on the map and then double click to finish
  • Results are given in miles
  • Use the dropdown box to select other units

Measure Location:

  • Select the “Location” button
  • Click on the map to calculate coordinates
  • Results are returned in a variety of coordinate systems




Query

Query data from active layers, search for specific records in a layer using the interactive query string builder.

To query a layer:

  • Click the query tool icon.
  • Select a layer (required) from the “Layer” dropdown menu. This layer is populated with the active layer list that is visible in the map.
  • Select an attribute field (required) from the selected layer in the “Field” dropdown menu.
  • Next, for the “Value” field, either select the “samples” radio button (when selected the dropdown list is populated with many samples of values contained in the attribute field) or select the “manual” radio button to manually enter the value.
  • Click the “Add to Query String” button to add the string to the query builder window. There is also an option to build a more complex query by using the “AND,” “OR,” and “NOT” buttons.
  • Click the “Execute Query” button to select the features that match the query string. Results will be displayed at the bottom of the Query Tool dialog box. Explore all of the records that matched the query string. Click “Zoom” to zoom to the map location for an individual record from the list of returned records.




Share Map Tool

Use this tool to share a custom map with others. The tool captures the active layers (in current stacking order) as well as the map extent and zoom level in a URL. Note that graphic elements are not preserved.

To Share the map:

  • Copy (Ctrl C or right mouse click and select copy) either the “Detailed Link” or the “Short Link”
  • Paste the URL into an email to share with others.
  • Paste the URL into another document to save to access the map at another time




Print

Print a custom map; add a title, author, and select from a list of layouts by downloading a PDF, PNG32 or JPG file to print.

To print the map:

  • The print tool will open a map preview window (this may take a moment)
  • Enter the Map Title, Author name, and select the desired layout
  • Click the “Download Your Map” button.  (Complex maps may take a few moments to load the output to a new browser tab.)




NOTE:

Pop-up Blockers maps prevent the printable map from opening in a new browser tab. If prompted, after clicking “Download Your Map”, simply select “Allow Pop-ups on this site.”

  • Once the printable map is viewable in a new browser tab, use “Save” (Ctrl S) to save a PDF or “Print” (Ctrl P) to send the map to a printer.
