Launch of maps.iu.edu:

I am quite proud to say that the ArcGIS Server based map application is now launched. This application is now the "official" campus map. I worked with the data owners in the University Architect's office and Facilities Management personnel to publish their data via ArcGIS server. Staff from the Office of Creative Services provided the icons and overall layout, but I am solely responsible for developing the application. This was a unique project in that the application resides on the IU Intellegent Infrastructure. This is a virtual server running ArcGIS Server and using the ArcGISFlex API. A PHP script checks for the client type on the page request, and if it is a mobile device, sends the user to the IU Mobile Maps application. IU Mobile will soon be incorporating this map service into their mobile map aplication! A preview of how this will work on their Google API based map is here.
